# **App Name**: Creator Canvas

## Core Features:

- Role-Based Onboarding: Mandatory onboarding flow for new users, tailored to different roles (Brand, Creator/Influencer, Service Provider/Freelancer, Studio), with progress tracking and skip functionality.
- Discovery & Content Feeds: Discovery feeds (/shorts, /reels, /explore) with content display, search, advanced filters, likes, comments, favorites, and sharing functionalities, including infinite scroll and optimistic UI updates.
- Unified Profiles: Unified profile view with role-based action buttons (Edit Profile, Follow/Unfollow, Favorite, Message, Collab Request), tabbed navigation, and a consistent layout for all user types.
- Real-Time Communication: Real-time chat functionality with unread indicators, favorites, team builder, and collab requests inbox, supporting direct messaging and group communication.
- Opportunity Marketplace: Project and opportunity marketplace for posting and finding gigs, supporting AI-generated descriptions and payment tracking.
- AI Job Description Generation: AI Job Builder tool generates structured job descriptions from text inputs. It functions as a tool for opportunity creation within the platform.
- AI-Powered Matchmaking: AI Matchmaking engine uses natural language processing to rank and recommend matches based on user input, displayed with profile cards.

## Style Guidelines:

- Primary color: Electric Purple (#BF5FFF) to reflect brand identity.
- Background color: Almost-White (#F5F0FF) to create a clean and modern look.
- Accent color: Neon Green (#A0F178) to highlight key interactive elements and calls to action.
- Font: 'GeistSans' (sans-serif) for all UI text, providing a modern and clean aesthetic. Note: currently only Google Fonts are supported.
- Use 'lucide-react' icons, with an outline style and consistent sizing, to maintain a modern and uniform visual language.
- Employ responsive breakpoints for optimal viewing on mobile, tablet, and desktop devices.
- Incorporate subtle micro-interactions like hover lift, smooth color transitions, and scale-down effects on tap to enhance user engagement.
